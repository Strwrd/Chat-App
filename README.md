# Chat-App
Example realtime chatting application using socket.io

### Getting Started : 
1. git clone https://github.com/Strwrd/Chat-App
2. yarn install
3. Done !

###Here some command 
#### Start App
````
yarn run start
````

####Start Chat-App at Development stage
````
yarn run devel
````

#### Run function test case
```` 
yarn run test
````

#### Run function test-cases with nodemon
````
yarn run test-watch
````
